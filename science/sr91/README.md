# SR-91 Aurora~II:	Formale Aerodynamik- und Leistungsanalyse eines Hochgeschwindigkeits-Aufklärungsflugzeugs

Diese Arbeit entwickelt und beweist formal die aerodynamischen, thermodynamischen, strukturmechanischen und pilotenphysiologischen Eigenschaften des fiktiven Hochgeschwindigkeits-Aufklärungsflugzeugs SR-91 AURORA II, das für eine Zwei-Piloten-Besatzung ohne zivile Passagiere ausgelegt ist. Das Flugzeug erreicht eine Höchstgeschwindigkeit von Ma = 5.0 bei einer Dienstgipfelhöhe von 30 km. Ein besonderer Schwerpunkt liegt auf der formalen mathematischen Herleitung der Wendeleistung im Überschallregime: Minimaler Kurvenradius, maximale Winkelrate und Lastvielfaches werden für das gesamte Flugzustandsgebiet analytisch nachgewiesen. Zwölf matplotlib-Abbildungen illustrieren die quantitativen Ergebnisse.
