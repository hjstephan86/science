# Die Ausdrucksstärke wissenschaftlicher Python-Bibliotheken: NumPy, SciPy, Matplotlib

Die Python-Bibliotheken **NumPy**, **SciPy** und **Matplotlib** bilden das technische Fundament nahezu aller modernen wissenschaftlichen Arbeiten, die auf Python basieren. Diese Arbeit gibt eine umfassende und
anschauliche Übersicht über die gesamte Ausdrucksstärke dieser drei Bibliotheken: Vom effizienten numerischen Rechnen mit mehrdimensionalen Arrays (**NumPy**) über höhere Algorithmen für Gleichungslöser, Signalverarbeitung, Statistik und wissenschaftliche Spezialfunktionen (**SciPy**) bis hin zur flexiblen, publikationsreifen
Datenvisualisierung (**Matplotlib**). Anhand zahlreicher kommentierter Codebeispiele und Abbildungen wird gezeigt, wie diese drei Werkzeuge im Verbund vollständige wissenschaftliche Arbeitsabläufe ermöglichen – von der Rohdatenverarbeitung bis zur fertigen Grafik für die Publikation.
