hawk

hawk — Bioinspiriertes Sturzangriffssystem (BioSAS).

Dieses Paket implementiert das aerodynamische Modell und den Regelungsalgorithmus des bioinspirierten Sturzangriffssystems, das auf den Flugeigenschaften des Wanderfalken (Falco peregrinus) basiert.

Öffentliche Schnittstelle

Aerodynamik: air_density, drag_coefficient, lift_coefficient, reference_area, drag_force, lift_force, terminal_velocity

Regelung: Phase, BGSGController, BioFalconTracker, proportional_navigation

121    """Bioinspiriertes Gleitflächen-Steuergesetz (BGSG) — Regler (Definition 5.3).
122
123    Der BGSG-Regler kombiniert einen Gleitflächen-Schaltanteil mit einem
124    PID-Äquivalentanteil:
125
126        s(t)       = ė(t) + Λ·e(t)                   (Gleitfläche)
127        u_sw(t)    = −K_sw · sign(s(t))               (Schaltanteil)
128        u_eq(t)    = −K_p·e − K_d·ė − K_i·∫e dτ     (PID-Äquivalent)

192        """Berechne den BGSG-Steuerausgang u_BGSG [m/s²].
193
194        Führt einen Regelschritt durch: Integralakkumulation, Gleitflächen-
195        berechnung, Schalt- und Äquivalentanteil sowie optionaler Vorsteuerung.
196
197        Parameters
198        ----------
199        error : array-like, shape (n,)
200            Lagefehler e = r_soll − r_ist [m].
201        error_dot : array-like, shape (n,)
202            Geschwindigkeitsfehler ė = v_soll − v_ist [m/s].
203        dt : float
204            Integrationsschrittweite Δt [s].
205        feedforward : array-like, shape (n,), optional
206            Externer Vorsteuerungsterm u_ff [m/s²], z. B. adaptive
207            Widerstandskompensation.

218        if self._integral.shape != error.shape:
219            self._integral = np.zeros_like(error)
220
221        # Accumulate integral  ξ += e·Δt
222        self._integral = self._integral + error * dt
223
224        # Sliding surface  s = ė + Λ·e
225        s = self.sliding_surface(error, error_dot)
226

230        # PID equivalent term

232
233        u = u_eq + u_sw
234        if feedforward is not None:
235            u = u + np.asarray(feedforward, dtype=float)
236        return u

Der BGSG-Regler kombiniert einen Gleitflächen-Schaltanteil mit einem PID-Äquivalentanteil:

s(t)       = ė(t) + Λ·e(t)                   (Gleitfläche)
u_sw(t)    = −K_sw · sign(s(t))               (Schaltanteil)

202            Geschwindigkeitsfehler ė = v_soll − v_ist [m/s].
203        dt : float

207            Widerstandskompensation.
208
209        Returns
210        -------
211        np.ndarray, shape (n,)
212            Steuerbeschleunigungsvektor u_BGSG(t) [m/s²].
213        """
214        error = np.asarray(error, dtype=float)
215        error_dot = np.asarray(error_dot, dtype=float)
216
217        # Lazy re-initialise integrator if dimension changes
218        if self._integral.shape != error.shape:
219            self._integral = np.zeros_like(error)
220
221        # Accumulate integral  ξ += e·Δt
222        self._integral = self._integral + error * dt
223
224        # Sliding surface  s = ė + Λ·e
225        s = self.sliding_surface(error, error_dot)
226
227        # Switching term (discontinuous)
228        u_sw = -self.k_sw * np.sign(s)
229
230        # PID equivalent term
231        u_eq = -self.k_p * error - self.k_d * error_dot - self.k_i * self._integral
232
233        u = u_eq + u_sw
234        if feedforward is not None:
235            u = u + np.asarray(feedforward, dtype=float)
236        return u

 66    v_rel: np.ndarray,
 67    nav_gain: float = 4.0,

 69    """Berechne den Proportionalnavigations-Steuervektor für Phase P5 (Fangschlag).
 70
 71    Proportionalnavigation (PN) liefert den kommandierten Beschleunigungsvektor:
 72
 73        n_c = N' · v_c · (ω_λ × los_hat)
 74
 75    wobei v_c die Schließgeschwindigkeit und ω_λ die Sichtlinienrotationsrate ist.
 76    Das PN-Gesetz garantiert Zielerfassung, wenn N' ≥ 2·(v_target/v_drone) + 1
 77    (Satz 5.1). Für Optimalität gilt 3 ≤ N' ≤ 5.
 78
 79    Parameters
 80    ----------
 81    p_target : array-like, shape (3,)
 82        Zielposition [m].
 83    p_self : array-like, shape (3,)
 84        Eigene Drohnenposition [m].
 85    v_rel : array-like, shape (3,)
 86        Relativgeschwindigkeit v_target − v_self [m/s].
 87    nav_gain : float
 88        Navigationskonstante N' (Standard: 4.0).
 89

 91    -------
 92    np.ndarray, shape (3,)
 93        Kommandierter lateraler Beschleunigungsvektor n_c [m/s²].
 94    """
 95    p_target = np.asarray(p_target, dtype=float)
 96    p_self = np.asarray(p_self, dtype=float)
 97    v_rel = np.asarray(v_rel, dtype=float)
 98
 99    los = p_target - p_self
100    los_norm = float(np.linalg.norm(los))
101    if los_norm < 1e-12:        # already at target — no command
102        return np.zeros(3)
103
104    los_hat = los / los_norm
105
106    # Closing speed: positive when range is decreasing
107    v_closing = -float(np.dot(v_rel, los_hat))
108
109    # LOS rotation rate vector  ω_λ = (los × v_rel) / |los|²
110    omega_los = np.cross(los, v_rel) / (los_norm ** 2)
111
112    return nav_gain * v_closing * np.cross(omega_los, los_hat)

Berechne den Proportionalnavigations-Steuervektor für Phase P5 (Fangschlag).

Proportionalnavigation (PN) liefert den kommandierten Beschleunigungsvektor:

n_c = N' · v_c · (ω_λ × los_hat)

wobei v_c die Schließgeschwindigkeit und ω_λ die Sichtlinienrotationsrate ist. Das PN-Gesetz garantiert Zielerfassung, wenn N' ≥ 2·(v_target/v_drone) + 1 (Satz 5.1). Für Optimalität gilt 3 ≤ N' ≤ 5.

Parameters

p_target : array-like, shape (3,) Zielposition [m]. p_self : array-like, shape (3,) Eigene Drohnenposition [m]. v_rel : array-like, shape (3,) Relativgeschwindigkeit v_target − v_self [m/s]. nav_gain : float Navigationskonstante N' (Standard: 4.0).

Returns

np.ndarray, shape (3,) Kommandierter lateraler Beschleunigungsvektor n_c [m/s²].
