# sandbx

Eine strukturierte Sammlung von technischen Referenzen und akademischen Notizen für Informatik und verwandte Disziplinen.

## Inhalt

### 1. Command-Line Tools Reference (`cmds.md`)
Praktische Beispiele und Workflows für häufig verwendete CLI-Tools:

- **pyreverse** - UML-Diagramme aus Python-Code generieren (DOT → PDF)
- **cairosvg** - SVG-Batch-Konvertierung zu PDF
- **ocrmypdf** - OCR-Texterkennung für gescannte PDFs
- **wkhtmltopdf** - HTML zu PDF Konvertierung mit Formatierung
- **gpg** - Verschlüsselung und Schlüsselverwaltung (inkl. Behördenschlüssel)

### 2. Vorlesungsnotizen (`notes.md`)
Chronologisch sortierte Mitschriften aus 25+ Vorlesungen, u.a.:

**Informatik & Software Engineering:**
- Parallel and Distributed Computing (Cray-1, Pipelining)
- Entwurf mikroelektronischer Systeme (Moore's Law, VHDL, Xilinx)
- Natural Language Processing (ML Pipeline, Overfitting)
- Autonome Robotermanipulation (Euler-Winkel, Transformationsmatrizen)
- Kollaborative Robotik (Cobots, KI-Sicherheit)

**Mathematik & Statistik:**
- Stochastic Analysis, Random Graphs
- Grundlagen der Statistik (Bayes, Wahrscheinlichkeitsräume)
- Statistische Datenanalyse mit R
- Funktionentheorie (Komplexe Differenzierbarkeit)

**Interdisziplinär:**
- Grundlagen der Genomforschung (KEGG, Metabolome)
- Data Science in der Biologie (Python/R-Integration)
- Algorithmic Cheminformatics (Wiener Index, Siedepunkte)
- Mathematische Biologie (Geometrische Verteilung, Hypothesentests)

## Verwendungszweck

- **Schnellreferenz** für wiederkehrende CLI-Workflows
- **Prüfungsvorbereitung** durch strukturierte Vorlesungszusammenfassungen
- **Wissensarchiv** für interdisziplinäre Konzepte
- **Nachschlagewerk** für mathematische und statistische Methoden

## Format

Alle Dokumente sind in Markdown formatiert für:
- Einfache Versionskontrolle mit Git
- Plattformunabhängige Lesbarkeit
- Konvertierung zu PDF/HTML bei Bedarf
- Integration in Wissensmanagementsysteme

## Suchhinweise

Nutze die Markdown-Überschriften zur Navigation:
- `##` für Vorlesungstermine/Tool-Namen
- `###` für Unterthemen
- `**Fettdruck**` für Schlüsselbegriffe
