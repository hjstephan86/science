# Subgraph Algorithmus und optimale Polygon-Tessellierung in der Echtzeit-Computergrafik

Der Subgraph Algorithmus ist ein effizienter Algorithmus zum Vergleichen zweier Graphen $G$ und $G'$ mittels Adjazenzmatrizen und Signatur-Arrays. Der Algorithmus bestimmt durch zyklische Rotation, ob ein Graph als Subgraph in einem anderen enthalten ist. Diese Arbeit wendet den Subgraph Algorithmus an für die optimale Polygon-Tessellierung in der Echtzeit-Computergrafik.
