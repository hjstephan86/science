# DNA-basierte Datenspeicherung mittels des Subgraph Algorithmus

Die digitale Datenmenge der Menschheit wächst exponentiell: Prognosen zufolge werden bis 2040 über 200 Zettabyte ($2 \times 10^{23}$ Byte) an Daten gespeichert sein müssen. Konventionelle Speichermedien --
magnetische Festplatten, Flash-Speicher, optische Medien -- stoßen dabei an physikalische, energetische und ökonomische Grenzen. Die Natur bietet eine fundamentale Alternative: Die Desoxyribonukleinsäure (DNA) speichert die gesamte genetische Information aller Lebewesen in einer Dichte von rund $2{,}15 \times 10^{15}$ Byte pro Gramm -- das entspricht etwa $215$ Exabyte pro Gramm, oder dem Vielfachen aller bisher produzierten digitalen Daten in einem Kaffeelöffel.
